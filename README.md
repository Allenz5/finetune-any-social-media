# Finetune Social Media

Chrome extension that shapes your LinkedIn feed with an AI agent.

## Develop

```bash
npm install
npm run dev      # build + watch into dist/
```

Then load `dist/` as an unpacked extension:

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked** and select the `dist/` folder

## Build

```bash
npm run build
```

Outputs a production bundle to `dist/`.
